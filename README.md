# 🌟 Personal Growth Tracking System (Console Application)

> A Java Console-Based Personal Growth Tracking System that helps users manage their daily habits, tasks, projects, and productivity while tracking XP and levels.

---

## 📖 Overview

LifeOS 365 is a menu-driven Java console application developed to help users organize their daily activities and monitor their personal growth.

The application provides a simple interface to manage habits, daily tasks, and projects while rewarding users with an XP and Level system to encourage consistency and productivity.

This project demonstrates the implementation of Object-Oriented Programming (OOP), CRUD operations, ArrayList, loops, methods, and menu-driven programming in Java.
